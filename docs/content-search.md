# Content Search

Content Search provides powerful search capabilities across objects and their associated files.

## Overview

The search system enables:
- Full-text search
- Metadata search
- File content search
- Advanced query options

## Key Benefits

1. **Discovery**
   - Find relevant content
   - Explore relationships
   - Discover patterns

2. **Integration**
   - Combined search results
   - Unified interface
   - Rich filtering

3. **Performance**
   - Optimized indexing
   - Fast results
   - Scalable search
