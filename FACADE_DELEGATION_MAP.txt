OBJECTENTITYMAPPER FACADE - METHOD DELEGATION MAP
==================================================

INHERITED FROM QBMapper (NO DELEGATION NEEDED):
- find() - Find by ID
- findAll() - Find all entities  
- findEntity() - Find single entity
- findEntities() - Find multiple entities
- delete() - Delete entity (we override with events)
- insert() - Insert entity (we override with events)
- update() - Update entity (we override with events)

DELEGATED TO HANDLERS:

1. LockingHandler:
   - lockObject()
   - unlockObject()

2. QueryBuilderHandler:
   - getQueryBuilder()
   - getMaxAllowedPacketSize()

3. CrudHandler:
   - Basic insert/update/delete (wrapped with events in facade)

4. StatisticsHandler:
   - getStatistics()
   - getRegisterChartData()
   - getSchemaChartData()
   - getSizeDistributionChartData()

5. FacetsHandler:
   - getSimpleFacets()
   - getFacetableFieldsFromSchemas()

6. BulkOperationsHandler:
   - ultraFastBulkSave()
   - deleteObjects()
   - publishObjects()
   - depublishObjects()
   - publishObjectsBySchema()
   - deleteObjectsBySchema()
   - deleteObjectsByRegister()
   - processInsertChunk()
   - processUpdateChunk()
   - calculateOptimalChunkSize()

7. QueryOptimizationHandler:
   - separateLargeObjects()
   - processLargeObjectsIndividually()
   - bulkOwnerDeclaration()
   - setExpiryDate()
   - applyCompositeIndexOptimizations()
   - optimizeOrderBy()
   - addQueryHints()
   - hasJsonFilters()

KEPT IN FACADE (ORCHESTRATION):
- insert() - with ObjectCreatingEvent/ObjectCreatedEvent
- update() - with ObjectUpdatingEvent/ObjectUpdatedEvent
- delete() - with ObjectDeletingEvent/ObjectDeletedEvent
- isRbacEnabled()
- isMultiTenancyEnabled()
- MultiTenancyTrait methods

TOTAL METHODS IN FACADE: ~25-30 (down from 68+)
